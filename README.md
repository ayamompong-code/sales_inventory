# Sistem Informasi Penjualan Alat Tulis Kantor

Project Akhir Mata Kuliah Web Service  
Menggunakan PHP, MySQL, dan REST API.

## Fitur
- REST API (Product, Category, Customer, Supplier, Stock)
- Database relasional (>5 tabel)
- Aplikasi web konsumsi API
- Manajemen user & role

## Cara Menjalankan
1. Import database `pos.sql` ke MySQL
2. Jalankan XAMPP (Apache & MySQL)
3. Akses:
   http://localhost:8080/sales_inventory

## API Endpoint
- GET /api/product.php
- GET /api/category.php
- GET /api/customer.php
- GET /api/supplier.php
